Student Score Predictions

Implemented a Flask-based web application for predicting student scores based on demographic and academic parameters. Users can input details such as gender, ethnicity, parental education level, lunch type, and test preparation status along with writing and reading scores. Utilized a custom data pipeline to preprocess input data, including standardization using StandardScaler. Integrated a prediction pipeline to generate predictions using a machine learning model in real-time. Enhanced user interaction through dynamic rendering of predicted scores on a web interface.

Book Genre Classifications

Developed a book genre classification system using natural language processing techniques and machine learning models. Preprocessed text data from book summaries, including text cleaning, removal of stopwords, lemmatization, and stemming using NLTK. Applied CountVectorizer to convert text data into numerical features and experimented with different configurations for optimal feature extraction. Trained and evaluated various classifiers such as Decision Tree, Random Forest, Naive Bayes, and Support Vector Machine (SVM) to predict book genres based on summarized content. Visualized word frequency distributions to gain insights into the most frequent words per genre category, enhancing model interpretability and performance.

Laptop Price Predictions

Developed a web-based application using Streamlit for predicting laptop prices based on various features. Utilized a Random Forest model trained on a dataset containing laptop specifications such as brand, type, RAM, operating system, weight, touchscreen availability, IPS display, screen size, resolution, CPU, HDD size, SSD size, and GPU brand. Implemented preprocessing steps including feature selection, data cleaning, and encoding categorical variables. Integrated the model into a user-friendly interface where users can input laptop specifications and receive predicted price ranges. Enhanced user experience with interactive dropdowns and input fields for real-time predictions.

Cryptos Buy Signals App

Developed a Streamlit-based application for automated trading signals in cryptocurrencies. Integrated with Binance API to fetch real-time candlestick data and calculate key technical indicators including Exponential Moving Averages (EMAs) and Relative Strength Index (RSI). Implemented multiple trading strategies based on EMA convergence and RSI thresholds to generate buy and sell signals. Enhanced user interface with customizable parameters for selecting cryptocurrency symbols, timeframe, initial investment, and trading strategy. Visualized candlestick charts with annotated trade signals using Plotly for interactive analysis. Evaluated trading performance with simulated trades, displaying trade history and final balance metrics.
